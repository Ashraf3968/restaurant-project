# Aurelio House Next.js Demo

Premium multi-page restaurant website built with Next.js 14 App Router and Tailwind CSS.

## Routes

- `/`
- `/about`
- `/menu`
- `/contact`
- `/reservation`
- `/login`
- `/signup`
- `/admin`
- `/privacy`
- `/terms`

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
